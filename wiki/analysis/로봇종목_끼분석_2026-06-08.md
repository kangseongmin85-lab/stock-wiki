## "끼" 판정 기준
종목의 **급등락 기질(탄력)**. 점수 ↑ 요소: ① 소형 시총 ② 저유통주식 ③ 순수 로봇테마(본업=로봇) ④ 과거 상한가/급등 이력 ⑤ 가벼운 거래량. 점수 ↓: 대형주·지주사·본업 따로(곁가지).
- 🔥🔥🔥 **초강끼**: 테마 점화 시 상한가·더블 가능. 변동성 극심·고위험
- 🔥🔥 **강끼**: 단기 20~30% 급등락 잦음
- 🔥 **중끼**: 테마 따라 움직이나 탄력 제한
- 💧 **저끼**: 대형·지주·곁가지. 테마로 크게 안 움

> ⚠️ 끼는 "기질"이지 "방향"이 아님. 끼 강할수록 급등도 급락도 큼 → 추격 금물, 손절 필수. 실제 진입은 질문 시점 실시간 가격·거래량 재확인([[analyze-at-question-time]]).

## 🎯 "끼 최상위" 워치리스트 (테마 재점화 시 탄력 1순위)

> 초강끼(🔥🔥🔥) + 순수 로봇테마 + 실체 재료 교집합

1. **에스비비테크** — 감속기 국산화 대장끼, 부품주 중 재료·이력 최강
2. **레인보우로보틱스** — 삼성 백업 휴머노이드, 변동성 극대
3. **로보티즈 / 뉴로메카** — 소형 순수 로봇 폭발형
4. **하이젠알앤엠 / 에스피지** — 액추에이터 후행수급 핵심
5. **씨메스 / 에스오에스랩 / 넥스트칩** — 피지컬AI 눈(비전·라이다) 소형 고변동
6. **휴림로봇 / 재영솔루텍** — 초저가 작전성 끼(고위험·단타영역)

## ⚠️ 끼 운용 원칙 (RULES 정합)
- 끼 강할수록 **추격 절대 금물** — 6/2 고점 후 이미 -17%(LG전자) 사례. 끼주는 조정도 그만큼 깊음
- **분할 진입 + 손절선 필수**. 적자·신규상장 끼주(레인보우·뉴로메카·나우·클로봇)는 비중 축소
- 선결조건: 코스피·미 SOX 매크로 반등(6/5 -5.54% 충격 진정). 매크로 안정 전 끼는 "급락 방향"으로도 터짐
- 부품주(②)가 완제품(①)보다 실적 가시성↑ → 같은 끼라도 **②그룹이 위험 대비 매력 우위**

### 위치로 갈리는 4가지 상태
- **🅰️ 아직 주도주(거래대금 1,500억+)**: 레인보우·로보티즈. 단 레인보우 PER 9,531배·로보티즈 저점대비 5.6배 = 극단 고밸류. 추격 금물, 지지 확인 후 분할.
- **🅱️ 박살난 낙주(저점권+거래대금 메마름)**: 하이젠알앤엠(-72%)·에스오에스랩(-65%)·씨메스(저점권)·재영솔루텍(-78%). 반등 탄력 최대지만 바닥 미확인 = 떨어지는 칼날. 거래대금 회복 전 관망.
- **🅲️ 부품주 중간**: 에스피지(거래대금 450억 유일 생존, 부품 수급 1순위 확인)>에스비비테크(62억, 재료대비 수급식음)>뉴로메카.
- **💀 저가 단타**: 휴림로봇(거래량 569만주·등락 -2%)·넥스트칩. 중기 투자 부적합.

### 결론
1. 로봇 테마 **전체 투매 진행** = 매크로 충격 미종료. 오늘은 **전 종목 매수 부적합**.
2. 살아있는 수급은 레인보우·로보티즈 딱 둘. 나머지 수급 실종.
3. 낙폭과대 반등 후보(하이젠·에스오에스랩·씨메스)는 **거래대금 회복**이 진입 신호.
4. 부품주는 에스피지가 체력 우위.
5. 선결조건: 코스피·미 SOX 안정. → 관망 + 거래대금 회복 종목 선별 국면.

> **판단 근거:**
> 1. 지수는 명백한 **코스피-코스닥 디커플링**. 외인 7조 매도는 환율(1,530)·금리 급등과 맞물린 구조적 자금 이탈 신호 — 대형주(특히 외인 보유 비중 높은 반도체·지주·자동차)는 추세 베팅이 위험.
> 2. 코스닥 반등은 펀더멘털이 아니라 **금융위 코스닥 활성화 긴급회의 + 국민성장펀드 기대**라는 정책 수급 단발 이벤트. 외인이 빠지는 자리를 기관 정책자금이 메운 "수급 공백 메우기"성격 → 지속성 검증 안 됨.
> 3. **백데이터 경고(결정적)**: 직전 동일 국면(반도체 소부장 급등일)인 5/22·5/26 cohort의 outcomes.json T+5는 주성엔지니어링 -17.2%, 에스에이엠티 -10.5%, SFA반도체 -22.6%, 아모텍(5/26 재추격) -28.0% 등 **대부분 -20~-36%**. 소부장 상한가 속출 → 며칠 뒤 급락이 직전 패턴.
> → 결론: **적극(공격) 매수 국면 아님.** 외인 매도·고환율이 멈추기 전까지는 코스닥 단기 모멘텀만 짧게, 작은 비중으로. 추격 매수는 백데이터상 손실 확률이 높음.

### 3. 종목 시나리오 (내일 6/5 매매 후보)

> 백데이터(outcomes.json)는 종목코드|과거flag일 기준. T+5는 "그날 스크리너에 잡힌 뒤 5거래일 수익률". 백데이터는 강한 경고 신호로 활용.

#### 후보 1. 원익IPS / 주성엔지니어링 (반도체 소부장 대장) — 정책수급+상한가 모멘텀
- **재료 등급**: 단기 이슈 (정책 긴급회의 + 코스닥 부양 기대). 구조적(HBM 전공정)과 혼재되나 당일 급등은 단기 성격
- **백데이터**: 주성엔지니어링(036930) **5/22 등장 → t3 -7.1% / t5 -17.2% / MFE +11.6% / MAE -19.1%**. 직전 추격이 손실로 끝남. 원익IPS는 outcomes 직접 이력 없음(기사기반)
- **재등장 여부**: 주성엔지니어링 13일 만에 재등장(06-04, +27.22%) — 모멘텀 재점화이나 백데이터상 단기 천장 가능성
- **매수 시나리오**: 신규 추격 비권장. 굳이 본다면 상한가 다음날 **갭 소화 후 눌림 지지 확인 시 소량**, 시초가 추격 금지
- **손절 기준**: 전일 상한가 시가(저점) 하향 이탈 시 즉시
- **리스크**: 외인 매도장에서 정책 기대 단발 소진 시 급반락. 직전 2회(5/22·5/26) 모두 T+5 -17~-23%

#### 후보 2. 삼성중공업 (조선/FLNG) — KB 4조 미국 델핀 FLNG PF 완료
- **재료 등급**: 중기 모멘텀 (미국 에너지 인프라 + 한미 조선협력, 수주 가시성)
- **백데이터**: outcomes.json 직접 이력 없음 → **기사기반. 차트·거래량 직접 확인 필요**
- **재등장 여부**: daily_signal 미등장(첫 부각)
- **매수 시나리오**: docx 언급 19건 최상위권 + 외인 매도장에서 수출/수주 모멘텀은 상대적 방어. **전고 돌파 시 거래량 동반 확인 후** 분할 접근
- **손절 기준**: 직전 박스권 하단 이탈 시
- **리스크**: 코스피 대형주 → 외인 7조 매도 지속 시 지수 동반 하락 압력. FLNG는 장기 프로젝트로 단기 실적 반영 제한

#### 후보 3. 엘앤에프 (LFP 양극재) — 국민성장펀드 2,200억 확보
- **재료 등급**: 단기 이슈~중기 (정책자금 + LFP 국산화). 양산은 3분기말로 시차
- **백데이터**: outcomes.json 직접 이력 없음 → **기사기반**
- **재등장 여부**: daily_signal 미등장
- **매수 시나리오**: docx 25건 언급(이차전지 1위). 정책 재료 당일 반영 여부 확인 후 **눌림 대응**. 추격 자제
- **손절 기준**: 재료 발표 당일 저가 이탈 시
- **리스크**: 전기차 캐즘·배터리 수출 부진(충북 47% 급감 기사) 구조적 역풍. 정책 기대만으로는 추세 약함

#### 후보 4. 현대차 (젠슨황 방한 / 로봇·자율주행) — 6/5 입국 모멘텀
- **재료 등급**: 단기 이벤트 드리븐 (7일 회동) + 구조적(AI·로봇 가치 재평가)
- **백데이터**: outcomes.json 직접 이력 없음(현대차 코드 005380 미수록) → **기사기반**. 단, 같은 군의 두산(000150) 5/29 등장 t1 +11.7%로 출발했으나 이후 데이터 미완(추세 확인 안 됨)
- **재등장 여부**: daily_signal 미등장(대형주)
- **매수 시나리오**: 젠슨황 입국(6/5)~회동(7일)이 분기점. **"sell the news" 경계** — LG전자 -22%가 이미 선반영 소진 사례. 회동 당일 재료 노출 후 차익실현 패턴 주의. 이벤트 베팅은 소량·짧게
- **손절 기준**: 입국 당일 시가 이탈 또는 회동 다음날 음봉 시 청산
- **리스크**: 외인 최대 매도 대상 대형주. 기대 선반영 → 뉴스 소멸 시 급락(젠슨황株 당일 와르르 사례)

#### ⚠️ 회피 권고 종목군
- **로봇/휴머노이드(레인보우로보틱스·로보스타·에스피지)**: overview 5/29 대장이었으나 재료소멸 차익실현 진행(-6~-12%). 반등 시도는 데드캣 위험.
- **한화에어로스페이스/한화**: 대전 폭발 압수수색·전국 9개 사업장 중단. 하반기 실적 기대보다 단기 악재 우세.

### 4. 유사 국면 참조 (패턴 엔진)

> situations.json + outcomes.json 결합. 오늘(반도체 소부장 정책 급등) 벡터와 가장 닮은 과거 = **2026-05-22**(반도체-장비 44.5·전력반도체·MLCC·테슬라 주도, 소부장 상한가 cohort).

| 유사 날짜 | 닮은 점 | 그날 대장 | 그날 cohort T+5 결과 |
|-----------|---------|-----------|----------------------|
| 2026-05-22 | 반도체 소부장 상한가 급등(주성·SFA·아모텍) | 주성엔지니어링(+20.95%) | 주성 **-17.2%** / SFA반도체 -22.6% / 한솔홀딩스 -2.1% / 아이엠티 -10.2% — **대부분 급락** |
| 2026-05-26 | 소부장 2차 + 우주/보안 | 아모텍 외 | 아모텍 **-28.0%** / 에스에이엠티 -10.5% / 대부분 cohort T+5 -20~-36% |
| 2026-05-29 | 로봇·스마트팩토리·엔비디아(젠슨황 테마 1차) | 두산/LG전자 | 두산 t1 +11.7% 후 데이터 미완. 로봇주 이후 차익실현으로 연결 |

> **해석**: 직전 두 차례(5/22·5/26) 소부장 상한가 국면은 **예외 없이 T+5에 큰 손실**로 귀결됐다. 오늘이 3번째 동일 패턴 → 추격 매수의 통계적 기대값이 명백히 음(-). "상한가 속출 = 단기 천장 신호"로 해석하고, 신규 진입보다 보유분 차익실현·관망이 합리적.

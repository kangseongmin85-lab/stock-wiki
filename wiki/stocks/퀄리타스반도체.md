# 퀄리타스반도체

## 한줄 요약

초고속 인터커넥트 반도체 IP(PCIe·UCIe·MIPI) 설계 전문. 삼성 파운드리 4~8nm IP 파트너로 美 베리실리콘·中 판세미에 IP 공급. 실리콘포토닉스·칩렛(UCIe) IP로 AI/CPO 밸류체인 탑승.

---

## 현재 투자 포인트

- 🟢 [강세] 삼성 파운드리 IP 생태계 핵심 파트너, 삼성 22.7조 장기 파운드리 수주 수혜 / UCIe 2.0 칩렛·실리콘포토닉스 IP / 재료등급: **구조적 성장**
- 🟢 [강세] 美 베리실리콘(PCIe6/CXL3 IP)·中 판세미(다회차 라이선스) 등 글로벌 고객 확보, 2026 흑자전환 목표
- 🔴 [리스크] **영업적자(IP 투자기)** — 매출 성장성으로만 평가. 수주 기사마다 급등락 큰 테마성 변동성

## 대장주 판단

국내 유일 UCIe PHY IP 보유, 초고속 인터커넥트 IP 니치 1군. 광통신 대장(오이솔루션)·HBM 대장(한미반도체)과 달리 **칩렛·인터커넥트 IP** 측면 수혜.
이 종목 포지션: 삼성 파운드리 IP 파트너 → 변화 조건: IP 라이선스 양산 로열티 본격화, 흑자전환

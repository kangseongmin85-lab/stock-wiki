# AI 데이터센터 / AI 팩토리 (밸류체인 허브)

> 이 페이지는 AIDC(AI Data Center) 밸류체인 **전체 지도**입니다. 각 분야 상세는 하단 세부 테마 페이지로 연결됩니다.
> 출처: NVIDIA GTC 2026(San Jose 3월) / GTC Taipei 2026(6월) 키노트 + 국내 증권가 섹터 리포트 / 갱신 2026-06-09

## 요약 — "데이터센터 → AI 팩토리"

젠슨 황이 GTC 2026에서 던진 핵심 메시지: **데이터센터는 이제 '전기를 토큰으로 바꾸는 공장(AI Factory)'이다.** 고객은 더 이상 "컴퓨터를 사는" 게 아니라 "AI 팩토리를 짓는다". 이 패러다임 전환이 밸류체인 전체의 투자 논리를 바꿨다.

**새로운 핵심 지표 = `tokens per watt` (와트당 토큰).**
AI 팩토리는 본질적으로 **전력 제약(power-constrained) 시스템**이다. 전력은 한정돼 있으므로, 같은 1MW로 얼마나 많은 토큰(=가치)을 뽑아내느냐가 경쟁력을 결정한다. → 그래서 **전력·냉각·네트워킹의 "효율"**이 GPU 성능만큼 중요해졌고, 여기서 한국 밸류체인의 기회가 열린다.

GTC 2026에서 젠슨 황이 직접 강조한 4대 축:
1. **전력 (Power)** — 800V HVDC 전환, UPS/PDU 제거, 1MW 랙 시대
2. **네트워킹 (Networking)** — 공동패키징광학(CPO)·실리콘포토닉스로 "구리의 벽" 돌파
3. **냉각 (Cooling)** — 공랭 → 수랭/액침 전면 전환
4. **컴퓨트 (Compute)** — Vera Rubin / HBM4 / DSX(AI팩토리 디지털트윈)

## 🗺️ 밸류체인 8대 분야 지도

> 자금(병목)은 **상류(반도체)에서 하류(전력·냉각·네트워킹)로 이동** 중. HBM·패키징 절대부족 구간을 지나 2026년은 **전력 인프라·액체냉각·광 네트워킹**으로 관심 이동.

|---|------|-----------|-----------------|------------|

2. [[stocks/두산에너빌리티]] — AI 전력 대란 해결사. 원전·SMR 주기기 독점 + 가스터빈 슈퍼사이클
3. [[stocks/HD현대일렉트릭]] — 초고압 변압기 수출 폭발. 미국 전력망 교체 + 800V HVDC 수혜
4. [[stocks/한미반도체]] — HBM TC본더 독점. 후공정 장비 대장
5. [[stocks/삼성전자]] — HBM4 + 파운드리 AI칩 + FC-BGA 기판, 풀스택
6. [[stocks/효성중공업]] / [[stocks/LS ELECTRIC]] — 전력기기 2~3선

## 📈 투자 관점 — 병목 이동 타임라인

```
2023      ChatGPT → AI 인프라 사이클 개시 (GPU 부족)
2023~24   HBM 절대부족 → SK하이닉스·한미반도체 폭발
2024~25   미국 전력망 노후 + AI 전력 급증 → 변압기(HD현대일렉트릭) 폭발
2025      딥시크 충격(효율화 논란) → 단기 조정 후 재상승
2025~26   ▶ 현재: 전력(800V HVDC)·액체냉각·광네트워킹(CPO)으로 병목 이동
2026~27   ▶ Vera Rubin / Kyber 랙(600kW~1MW) 양산 → 전력·냉각 2차 수혜 본격화
          ▶ HBM4 양산, 1.6T 광트랜시버, CPO 상용화
```

핵심 통찰: **"GPU가 좋아질수록 → 더 많은 전기·더 강한 냉각·더 빠른 연결이 필요"** → 하류 인프라가 구조적으로 따라 커진다. 이것이 한국(전력기기·HBM·냉각·광통신 경쟁력 보유)의 구조적 수혜 논리.

## ⚠️ 유의 사항
- **CAPEX 사이클 리스크**: 빅테크 데이터센터 투자가 꺾이면 전력·냉각·네트워킹주 동반 타격
- **딥시크류 효율화**: AI 모델 효율 급상승 시 HBM·GPU 수요 추정 하향 가능
- **미-중 반도체 규제**: 수출 통제 변동
- **수혜 범위가 너무 넓음** → 테마 편승 부실주 옥석 가리기 필수. 실제 수주·납품 레퍼런스 확인
- **차트 확인 필수**: 본 페이지는 재료 지도일 뿐, 진입은 거래량·차트로 별도 검증
